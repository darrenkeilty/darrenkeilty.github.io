UCalgary Marketplace

Full-stack marketplace platform for University of Calgary students

Project Overview

By Darren Keilty, Enioluwafe Balogun, Deepshikha Dhammi, Sarim Ekram, Imdad Goraho
SENG 513 - Web-Based Systems

UCalgary Marketplace is a web application built to give University of Calgary students one place to browse listings, post items and services, discover campus events, and manage interactions through a clean student-focused interface.

The project was developed as part of a Web-Based Systems course, and gave me the opportunity to develop a full-stack web application as part of a team, from scratch. The platform includes a React frontend, Express backend, and MySQL database, with Docker Compose for ease of development and deployment.

Key Features

  • Supports both marketplace posts and event posts in one student-focused platform.
  • Includes user account flows, post management, search, filtering, saved listings, and seller contact interactions.
  • Provides moderation workflows through reporting tools and an admin dashboard.
  • Built as a responsive, user-friendly application with a mobile-first user design and a desktop-first admin design.

Technical Highlights

  • Designed and implemented relational database structure to support shared post content, specialized post types, user and moderation actions.
  • Built REST-based backend functionality to connect front-end workflows with persistent data storage.
  • Worked across both UI design and systems structure, helping connect the user experience with the underlying engineering decisions.
